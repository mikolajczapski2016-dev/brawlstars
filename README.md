# Miki GameShop

Miki GameShop to mobilna platforma gier Mikołaja. Obecnie pierwszą dostępną grą jest Zombie Story, a kolejne gry mają przygotowane placeholdery.

## Motto projektu

**Full stack up to date, no technical debt.**

Pracujemy na najnowszych stabilnych wersjach zależności, narzędzi i runtime'ów. Jeśli zależność jest przestarzała, aktualizujemy ją zamiast budować nowe rzeczy na starym fundamencie.

## Mobile first

Telefon jest głównym urządzeniem dla tej platformy. UI, sterowanie, animacje, przyciski i pełny ekran projektujemy najpierw pod ekran dotykowy, a dopiero potem dopasowujemy do komputera.

## Komendy

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deploy

Projekt buduje się przez Vite i jest gotowy do publikacji na GitHub Pages z folderu `dist` przez workflow GitHub Actions.
